The Problem

The current market for smart gun sellers and collectors has no method of provenance for firearms.

No record taken for firearm history

There are currently no public or private records for each gun's lifecycle.

No standardized P2P network

There is no certified network for peer-to-peer transactions in the smart gun market.

Expensive and time consuming

Going through brokers and gun shows is expensive and inefficient for the occasional smart gun owner.

The solution

Revamping the peer to peer smart gun market by using Blockchain-based credentials

Validate sellers and purchasers

Proof of ownership is validated on the blockchain and transactions will onlu occur between valid and trusted parties.

Implement smart contracts for each transaction

By ensuring that the transaction occurring between the parties is legitimate every transaction is fair and trustworthy.

Store these transactions on a decentralized unit

Everyone's data is kept on a decentralized and immutable database.
